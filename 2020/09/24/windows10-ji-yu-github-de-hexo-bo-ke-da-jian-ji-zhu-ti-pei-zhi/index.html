Windows10基于github的Hexo博客搭建及主题配置

  • 小记：最近发现之前搭好的博客打不开了，花了好些时间解决bug,但是上网查了很多的博客依然没有解决，因此决定重新从零开始，借此机会将搭建过程写到博客里，以下为搭建hexo博客及设置主题的步骤与过程。

一.安装nodejs和git

  • git 下载: https://git-scm.com/

  • 下载好后，右击鼠标就会出现git的标志，这就表明git已经安装好

  • nodejs下载 :https://nodejs.org/en/download/

    (博主是windows系统，根据自己电脑系统进行选择)

  • 下载后双击node.exe安装即可

  • 在git终端查看node、npm和git版本以此来判断git和nodejs是否安装成功，如下图可知，git和nodejs安装成功

二.hexo博客搭建

  • 创建一个放置自己博客的文件夹（例如：Myblogs,这个目录就是博客的根目录），在Myblogs文件夹中右击鼠标，从中找到Git Bash Here选项，打开 git 窗口

  • 安装hexo（时间可能很长，耐心等待~）

    $ npm i -g hexo-cli    使用全局安装

    $ hexo init   初始化

    $ hexo g   生成博客目录文件

    $ hexo s   开启本地服务  Ctrl+C可取消预览

  • 在浏览中打开” http://localhost:4000"，可以看到博客已经在本地安装好

三.将博客同步到github上

  • 配置userName和email 

    $ git config --global user.name"userName"    userName 是github的用户名
$ git config --global user.email"yourEmail"  email是github的邮箱

  • 生成密钥

    $ cd ~/.ssh      查看是否有.shh文件夹
$ ssh-keygen -t rsa -C "yourEmail"    连续输入三个回车，生成密钥（密钥的存放路径是：C:\Users\Administrator.ssh） 

    $ eval "$(ssh-agent -s)"     添加密钥到ssh-agent
$ ssh-add ~/.ssh/id_rsa      添加生成的SSH key到ssh-agent

  • 创建仓库

    新建一个仓库，仓库名为：username.github.io.git（例如，你的用户名为test，那么就是test.github.io.git）、选择public即可，勾选创建README

  • 生成密钥

    点击右上角头像—>setting —> SSH and GPG keys —> SSH keys,新建一个SSH key 将 C:\Users\Administrator.ssh 下的 id_rsa.pub文件里的内容复制上去(注：.pub文件用windows自带的软件无法打开，可以用notepad++打开 )

    $ ssh -T git@github.com  #测试添加ssh是否成功。如果看到Hi后面是你的用户名，就说明成功了

  • 安装hexo部署工具

    $ npm i hexo-deployer-git --save
#修改站点配置文件_config.yml（在你搭建博客的文件夹中找到_config.yml文件，修改deploy（在末尾））
$ vi _config.yml    进入vim编辑器,i--进入插入模式，此时方可改动文件，移动时可通过鼠标或上下键，退出时用 先按Esc,再输入  :wq   保存退出

  • 将本地的博客上传到github上

    $ cd D:/MyBlogs   进入博客根目录
$ hexo g   生成文件
$ hexo s   打开本地服务器，在本地预览效果
$ hexo d   将博客上传到github上（可能需要等几分钟才能够在网页上看到更新后的效果）

  • 可能会出Deployer not found :git,此时输入如下命令即可解决

    $ npm install hexo-deployer-git --save

  • 此时可以看到github上生成了多个文件

  • 浏览器中输入username.github.io就可以浏览，如博主的 sourcex99.github.io

  • 至此，hexo博客的搭建以及关联github已经完成，接下来进行主题的设置

四.博客主题设置（以安装snark为例）

  • 找到你想要的主题 网址：https://hexo.io/themes/ 或 github上搜 hexo-themes

  • 找到到主题后，进入主题项目所在的地址 ：https://github.com/Litreily/hexo-theme-snark，根据README.md进行安装

  • 切换到博客的根目录 cd D:/MyBlogs

  • Installation

    #查看主题的路径，snark是https://github.com/litreily/hexo-theme-snark.git themes/snark,这里已经显示themes了，
#所以直接在根目录(如：D：/MyBlogs)里安装，如果没有这个，要切换到themes中进行安装 (cd /themes)
$ git clone https://github.com/litreily/hexo-theme-snark.git themes/snark
$ npm install hexo-renderer-pug --save
$ npm install hexo-renderer-sass --save
    #修改highlight
$ cd D:/MyBlogs      进入博客根目录
$ vi _config.yml     进入vim编辑器,找到highlight进行如下修改

  • 修改博客名字、作者、theme(注意，配置文件一定要修改相对应的主题名称，否则博客用的还是默认的主题),修改完了 Esc(进入末行命令) :wq （保存后退出）

  • 添加支持包

    $ hexo new page archives
$ hexo new page categories
$ hexo new page tags
$ hexo new page about

  • 安装插件

    $ npm install hexo-generator-feed --save      用于RSS
$ npm install hexo-generator-search --save    用于local search
$ npm install hexo-generator-sitemap --save   用于sitemap

  • 生成博客

    $ hexo clean      清除缓存
$ hexo g          生成博客      #一定要先生成
$ hexo d          关联仓库

    到这里，博客的主题就设置好了，在此基础上，博主对主题进行了优化
